### Installation

Dibaawah ini adalah cara untuk memasang aplikasi ini dalam komputer kamu :

1. Clone the repo
   ```sh
   git clone https://github.com/dikymonz/personal-notes
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Runn the code
   ```js
   npm run dev
   ```
